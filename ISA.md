# Simple CPU 指令集参考

## 指令集概述

Simple CPU 采用精简指令集（RISC）架构，共支持 **16条指令**，每条指令为 **32位** 定长格式。

```
指令格式:
31                 12  11   8  7    4  3    0
├─────────────────────┼───────┼───────┼───────┤
│   immediate/src_1   │ src_2 │ dest  │ opcode│
│       [19:0]        │ [3:0] │ [3:0] │ [3:0] │
└─────────────────────┴───────┴───────┴───────┘
```

---

## 指令分类

| 类别 | 指令数 | 说明 |
|------|--------|------|
| 数据传送 | 1条 | 立即数加载 |
| 算术运算 | 2条 | 加法、减法 |
| 逻辑运算 | 3条 | 与、或、异或 |
| 移位运算 | 2条 | 逻辑左移、逻辑右移 |
| 内存访问 | 2条 | 读内存、写内存 |
| 跳转指令 | 2条 | 直接跳转、寄存器跳转 |
| 分支指令 | 4条 | 相等、不等、小于、大于等于分支 |

---

## 指令列表

### 1. 数据传送指令

#### SET - 设置立即数

| 字段 | 值 |
|------|-----|
| 操作码 | `0000` (0x0) |
| 操作 | `R[dest] = immediate` |
| 说明 | 将20位立即数存入目标寄存器 |

**汇编示例：**
```
MOV R5, #100    # R5 = 100
```

---

### 2. 算术运算指令

#### ADD - 加法

| 字段 | 值 |
|------|-----|
| 操作码 | `0001` (0x1) |
| 操作 | `R[dest] = R[src_2] + R[src_1]` |
| 说明 | 将两个源寄存器的值相加，结果存入目标寄存器 |

**汇编示例：**
```
MOV R3, R1 + R2    # R3 = R1 + R2
```

#### SUB - 减法

| 字段 | 值 |
|------|-----|
| 操作码 | `0010` (0x2) |
| 操作 | `R[dest] = R[src_2] - R[src_1]` |
| 说明 | 将两个源寄存器的值相减，结果存入目标寄存器 |

**汇编示例：**
```
MOV R3, R5 - R2    # R3 = R5 - R2
```

---

### 3. 逻辑运算指令

#### AND - 按位与

| 字段 | 值 |
|------|-----|
| 操作码 | `0011` (0x3) |
| 操作 | `R[dest] = R[src_2] & R[src_1]` |
| 说明 | 对两个源寄存器进行按位与运算 |

**汇编示例：**
```
MOV R4, R7 & R3    # R4 = R7 & R3
```

#### OR - 按位或

| 字段 | 值 |
|------|-----|
| 操作码 | `0100` (0x4) |
| 操作 | `R[dest] = R[src_2] | R[src_1]` |
| 说明 | 对两个源寄存器进行按位或运算 |

**汇编示例：**
```
MOV R4, R7 | R3     # R4 = R7 | R3
```

#### XOR - 按位异或

| 字段 | 值 |
|------|-----|
| 操作码 | `0101` (0x5) |
| 操作 | `R[dest] = R[src_2] ^ R[src_1]` |
| 说明 | 对两个源寄存器进行按位异或运算 |

**汇编示例：**
```
MOV R4, R7 ^ R3    # R4 = R7 ^ R3
```

---

### 4. 移位运算指令

#### SLL - 逻辑左移

| 字段 | 值 |
|------|-----|
| 操作码 | `0110` (0x6) |
| 操作 | `R[dest] = R[src_2] << R[src_1]` |
| 说明 | 将 src_2 寄存器的值左移 src_1 位 |

**汇编示例：**
```
MOV R3, R4 << R1    # R3 = R4 << R1
```

#### SRL - 逻辑右移

| 字段 | 值 |
|------|-----|
| 操作码 | `0111` (0x7) |
| 操作 | `R[dest] = R[src_2] >> R[src_1]` |
| 说明 | 将 src_2 寄存器的值逻辑右移 src_1 位 |

**汇编示例：**
```
MOV R3, R4 >> R1    # R3 = R4 >> R1
```

---

### 5. 内存访问指令

#### MWR - 内存写

| 字段 | 值 |
|------|-----|
| 操作码 | `1000` (0x8) |
| 操作 | `Mem[R[src_1]] = R[src_2]` |
| 说明 | 将 src_2 寄存器的值写入 src_1 寄存器指向的内存地址 |

**汇编示例：**
```
MOV [R1], R5        # Mem[R1] = R5
```

#### MRD - 内存读

| 字段 | 值 |
|------|-----|
| 操作码 | `1001` (0x9) |
| 操作 | `R[dest] = Mem[R[src_1]]` |
| 说明 | 从 src_1 寄存器指向的内存地址读取数据到 dest 寄存器 |

**汇编示例：**
```
MOV R3, [R7]        # R3 = Mem[R7]
```

---

### 6. 跳转指令

#### JAL - 跳转并链接

| 字段 | 值 |
|------|-----|
| 操作码 | `1010` (0xA) |
| 操作 | `R[dest] = PC + 1; PC = immediate` |
| 说明 | 跳转到立即数指定的地址，并将返回地址存入目标寄存器 |

**汇编示例：**
```
JMP R6, #100      # R6 = PC + 1, PC = 100
```

#### JALR - 寄存器跳转并链接

| 字段 | 值 |
|------|-----|
| 操作码 | `1011` (0xB) |
| 操作 | `R[dest] = PC + 1; PC = R[src_1]` |
| 说明 | 跳转到 src_1 寄存器指定的地址，并将返回地址存入目标寄存器 |

**汇编示例：**
```
JMP R6, R10      # R6 = PC + 1, PC = R10
```

---

### 7. 分支指令

#### BEQ - 相等分支

| 字段 | 值 |
|------|-----|
| 操作码 | `1100` (0xC) |
| 操作 | `PC = (R[src_2] == R[dest]) ? R[src_1] : PC + 1` |
| 说明 | 如果 src_2 和 dest 寄存器的值相等，则跳转到 src_1 寄存器指向的地址 |

**汇编示例：**
```
BRC R10, R5 == R3   # if (R5 == R3) PC = R10 else PC = PC + 1
```

#### BNE - 不等分支

| 字段 | 值 |
|------|-----|
| 操作码 | `1101` (0xD) |
| 操作 | `PC = (R[src_2] != R[dest]) ? R[src_1] : PC + 1` |
| 说明 | 如果 src_2 和 dest 寄存器的值不等，则跳转到 src_1 寄存器指向的地址 |

**汇编示例：**
```
BRC R10, R5 != R3   # if (R5 != R3) PC = R10 else PC = PC + 1
```

#### BLT - 小于分支

| 字段 | 值 |
|------|-----|
| 操作码 | `1110` (0xE) |
| 操作 | `PC = (R[src_2] < R[dest]) ? R[src_1] : PC + 1` |
| 说明 | 如果 src_2 寄存器的值小于 dest 寄存器的值，则跳转到 src_1 寄存器指向的地址 |

**汇编示例：**
```
BRC R10, R5 < R3   # if (R5 < R3) PC = R10 else PC = PC + 1
```

#### BGE - 大于等于分支

| 字段 | 值 |
|------|-----|
| 操作码 | `1111` (0xF) |
| 操作 | `PC = (R[src_2] >= R[dest]) ? R[src_1] : PC + 1` |
| 说明 | 如果 src_2 寄存器的值大于等于 dest 寄存器的值，则跳转到 src_1 寄存器指向的地址 |

**汇编示例：**
```
BRC R10, R5 >= R3   # if (R5 >= R3) PC = R10 else PC = PC + 1
```

## 指令编码速查表

| 操作码 | 助记符 | 类型 | 功能描述 |
|--------|--------|------|----------|
| 0000 | SET | I-type | `R[dest] = immediate` |
| 0001 | ADD | R-type | `R[dest] = R[src_2] + R[src_1]` |
| 0010 | SUB | R-type | `R[dest] = R[src_2] - R[src_1]` |
| 0011 | AND | R-type | `R[dest] = R[src_2] & R[src_1]` |
| 0100 | OR | R-type | `R[dest] = R[src_2] \| R[src_1]` |
| 0101 | XOR | R-type | `R[dest] = R[src_2] ^ R[src_1]` |
| 0110 | SLL | R-type | `R[dest] = R[src_2] << R[src_1]` |
| 0111 | SRL | R-type | `R[dest] = R[src_2] >> R[src_1]` |
| 1000 | MWR | M-type | `Mem[R[src_1]] = R[src_2]` |
| 1001 | MRD | M-type | `R[dest] = Mem[R[src_1]]` |
| 1010 | JAL | J-type | `R[dest] = PC+1; PC = immediate` |
| 1011 | JALR | J-type | `R[dest] = PC+1; PC = R[src_1]` |
| 1100 | BEQ | B-type | `PC = (R[src_2]==R[dest]) ? R[src_1] : PC+1` |
| 1101 | BNE | B-type | `PC = (R[src_2]!=R[dest]) ? R[src_1] : PC+1` |
| 1110 | BLT | B-type | `PC = (R[src_2]<R[dest]) ? R[src_1] : PC+1` |
| 1111 | BGE | B-type | `PC = (R[src_2]>=R[dest]) ? R[src_1] : PC+1` |
